Simple web app that calculates the average grade and academic status of a student using JavaScript

## Description
- Student writes the `name` and `surnames`, and the `3 marks`
- The program `validates` the input data
- Calculates the `average` and shows the `academic qualification`

## Technologies
- HTML
- CSS
- JavaScript

- DOMContentLoaded event
- Form handling and events (`submit`, `reset`)
- Input validation
- DOM manipulation (`getElementById`, `textContent`)
- Separation of `HTML`, `CSS`, `JavaScript`
